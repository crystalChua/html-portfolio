Memory Game

During my internship in 2019, I worked on an app development project for BESTA Mart, a company that produces electronic dictionary devices widely used in Malaysia. I created a game, which I named 'Memory Game,' that was essentially a matching game.

Language: C++
IDE Software: Android Studio
Purpose: Kids can learn words while playing. Used colorful pictures to make the content more attractive.
Problem Solution
The back button stumped me for a long time as I wanted it to return directly to the main menu page when clicked, instead of navigating to the previous activity. I sought senior help and conducted extensive online searches. Finally, found the best solution, which involved using 'setFlags' and 'Intent' to target the specific activity.
